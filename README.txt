Word Birds v3 — 多词书 + 艾宾浩斯遗忘曲线版

使用方法:
1. 双击 index.html 运行游戏
2. 输入用户名登录（每个用户有独立记录）
3. 选择词书：小学 / 初中 / 高中 / 考研 / 雅思
4. 在词典中勾选想学的单词
5. 点击"开始游戏"进入游戏
6. 游戏中点击"结束学习"查看学习报告

功能:
- 5大词书（小学953 / 初中1799 / 高中4196 / 考研5490 / 雅思7896）
- 基于艾宾浩斯遗忘曲线的科学复习系统
- 记忆曲线面板（7天学习记录、遗忘曲线图、各级别分布）
- 个人错词本（跨词书汇总、按错误次数排序）
- 复习提醒横幅 / 英美双语发音 / 随机选词 / 学习模式预览
